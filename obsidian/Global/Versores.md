
# Versores

Versores são vetores unitários que apresentam a mesma orientação de um eixo.
![[Pasted image 20220311224858.png|250]]  

Representações comuns:

| $x$ | $y$ | $z$ |
| - | - | - |
| $\hat i$ | $\hat j$ | $\hat k$ |
| $\vec i$ | $\vec j$ | $\vec k$ |
| $\hat x$ | $\hat y$ | $\hat z$ |
